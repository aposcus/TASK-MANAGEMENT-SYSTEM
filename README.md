Zone Task Management System
Project Overview
The Zone Task Management System is a streamlined web application designed to help individuals and small teams organize their projects into sprints, track progress visually, and prioritize work efficiently. Built with a clean, modern interface, it provides immediate clarity on what tasks are 'To Do', 'In Progress', and 'Done'.

This project was developed as part of my learning journey at the SRM Institute of Science and Technology.

Features
Sprint-Based Organization: Organize tasks into manageable "Current Sprint" views.
Kanban-style Board: Drag-and-drop workflow visualization.
Task Prioritization: Color-coded priority labels (High, Medium, Low).
Progress Tracking: Visual progress bar showing completion percentage.
Task Management: Easy options to add, edit, and delete tasks.
User Profiles: Includes basic user sign-in and sign-up interfaces.
